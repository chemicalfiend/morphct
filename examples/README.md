# Examples

These example notebooks demonstrate the workflow employed in morphct.
To create the conda environment:
```bash
conda env create -f environment-ex.yml
conda activate morphct-ex
cd ../
pip install .
```

To run the examples:
```bash
cd examples
jupyter lab
```
